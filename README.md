# Money Tracker

## Description

This is a money tracker app that helps you to track your money and expenses.

## Features

- Add income and expense
- View income and expense
- Delete income and expense

## Technologies

- React
- Ionic
- Typescript
- React Hooks
- React Context
- React Router

## Installation

- Clone the repo

```bash
git clone

```

- Install dependencies

```bash
npm install

```

- Run the app

```bash
npm start

```
